Shorts Lens Privacy Policy / Shorts Lens 隐私政策

Last updated: April 28, 2026

English

Shorts Lens does not collect, sell, share, or transmit user data to any external server.

The extension runs only on YouTube pages. It reads YouTube Shorts page information, including video URL, title, view count, publish date, and video ID, in order to display metadata and save selected Shorts records locally.

All settings and collected Shorts records are stored locally in the user's Chrome browser using chrome.storage.local. The data is not uploaded to the developer or any third party.

Users can delete collected records from the extension's records page at any time.

Shorts Lens does not use analytics, advertising, tracking, remote code, or external APIs.

The use of information received from Chrome extension APIs complies with the Chrome Web Store User Data Policy, including the Limited Use requirements.

中文

Shorts Lens 不收集、出售、分享或传输用户数据到任何外部服务器。

本扩展仅在 YouTube 页面运行。它会读取 YouTube Shorts 页面中已有的信息，包括视频链接、标题、播放量、发布日期和视频 ID，用于显示视频元数据并将用户选择或符合规则的 Shorts 记录保存在本地。

所有设置和 Shorts 收集记录都通过 chrome.storage.local 保存在用户本机 Chrome 浏览器中，不会上传给开发者或任何第三方。

用户可以随时在扩展的记录页面删除已收集的记录。

Shorts Lens 不使用分析、广告、追踪、远程代码或外部 API。

本扩展对 Chrome 扩展 API 所获取信息的使用遵守 Chrome Web Store 用户数据政策，包括 Limited Use（有限使用）要求。
